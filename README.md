# Para iniciar

## Para instalar

```bash
yarn install
```

## Para iniciar DEV (LocalHost)

```bash
 yarn dev
```

## Para iniciar Build PROD

```bash
 yarn build
```

## Para iniciar Prod (Heroku)

```bash
 yarn start
```

## Ruta api /smallest POST en local

```bash
 http://localhost:3000/api/smallest
```
